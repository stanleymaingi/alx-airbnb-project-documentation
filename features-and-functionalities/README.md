## Project Overview

The ALX Airbnb Project is a web-based platform for property rentals that allows users to list, search, and book properties. The project simulates a real-world Airbnb system with multiple roles and functionalities.

---

## User Features

- **User Authentication**: Registration, login, and logout functionalities.  
- **Profile Management**: Update profile details, profile picture, and password.  
- **Search Properties**: Filter properties by location, price, and type.  
- **Property Details**: View detailed information including images, amenities, and reviews.  
- **Booking System**: Reserve properties for specific dates and manage bookings.  
- **Ratings & Reviews**: Rate properties and leave feedback for hosts.  

---

## Admin Features

- **Dashboard Access**: View platform analytics including total users, properties, and bookings.  
- **User Management**: Add, edit, or delete user accounts.  
- **Property Management**: Approve or remove property listings submitted by users.  
- **Booking Oversight**: Track and manage all bookings across the platform.  
- **Report Generation**: Export reports for user activity and property statistics.  

---

## Property Management

- **Add Listings**: Create new property listings with images, description, and pricing.  
- **Update Listings**: Edit property details or pricing dynamically.  
- **Delete Listings**: Remove properties from the platform.  
- **Availability Calendar**: Track availability to prevent double bookings.  

---

## Booking and Payments

- **Secure Booking**: Reserve properties with validation of dates and availability.  
- **Payment Integration**: Accept and process payments using mock gateways.  
- **Booking History**: Users can view their past and upcoming bookings.  
- **Cancellation Policies**: Manage cancellations and refunds based on set rules.
