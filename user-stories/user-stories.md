# User Stories for ALX Airbnb Project

## User Stories

1. **User Registration & Login**
   - As a user, I want to be able to register and log in so that I can access the platform and manage my bookings.

2. **Property Search**
   - As a user, I want to search for properties by location and availability so that I can find suitable accommodations.

3. **Property Booking & Payment**
   - As a user, I want to book a property and make a payment so that I can confirm my stay.

4. **Booking History**
   - As a user, I want to view my booking history so that I can track past and upcoming reservations.

5. **Property Management (Host)**
   - As a host, I want to add and manage my property listings so that guests can book my properties.

6. **Property Approval & Admin Oversight**
   - As an admin, I want to approve or remove property listings and manage users so that the platform maintains quality and security.

7. **Reports & Dashboard**
   - As an admin, I want to view reports and analytics so that I can monitor platform activity and performance.
