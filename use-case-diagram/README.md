# Use Case Diagram

This directory contains the Use Case Diagram for the ALX Airbnb Project, visualizing the interactions between system actors and key functionalities.

## Actors
- **User (Guest)**: Registers, searches properties, books properties, makes payments.
- **Host**: Adds and manages property listings.
- **Admin**: Approves/removes properties, manages users, oversees bookings.

## Key Functionalities
- User registration & login
- Property search & booking
- Payment processing
- Property management
- Admin oversight
